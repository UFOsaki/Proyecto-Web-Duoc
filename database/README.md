# Base de Datos — Sharingan Comics

## Motor
Oracle Autonomous Database (ATP) conectado via Oracle Wallet.

## Tablas
- **USUARIOS**: Usuarios registrados con password hasheado (BCrypt).

## Ejecución
1. Abrir SQL Developer.
2. Conectar usando la Wallet (`Wallet_sharingan.zip`).
3. Ejecutar `schema.sql` para crear la tabla.
4. (Opcional) Ejecutar `seed.sql` para datos de prueba.

## Notas
- **Nunca guardar contraseñas planas** — siempre usar BCrypt.
- La tabla usa `GENERATED BY DEFAULT AS IDENTITY` para auto-incremento.
- Los índices en EMAIL y USERNAME optimizan login y búsquedas.
